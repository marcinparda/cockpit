## Global Formatting Standards

### 2-Space Indentation
All files use 2-space indent with spaces (not tabs). Source: `.editorconfig`, `.prettierrc`.

### UTF-8 Encoding
All files encoded in UTF-8. Source: `.editorconfig`.

### Final Newline Required
All files must end with a newline character. Source: `.editorconfig` insert_final_newline = true.

### No Trailing Whitespace
Trim trailing whitespace from all lines. Exception: Markdown files (`.md`) preserve trailing whitespace. Source: `.editorconfig`.
